# Invoice-Downloader

Wordpress woocommerce bulk invoice downloader plugin, It will allow user to download all the orders as invoices pdf wrapped in a zip file. There will be a download button appear at the top of order page after activate this plugin

## Setup

- Clone the repository into the plugins directory of your wordpress website
- after clone the repository go to wp-admin -> plugins section and activate the plugin "Invoice-Downloader"
- after activate the plugin go to woocommerce -> orders page and see if there a download button appear at the top of the orders table
